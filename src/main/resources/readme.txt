Dear SimpleBroadcast user,

first of all: Thank you for using my plugin.
In this readme some very important things get approached which you should regard while using the plugin:

- The prefix, suffix as well as all messages have to be surrounded by single quotes ('...') in order that
  the colour codes and also the variables can work properly.
- It's not possible to write out single quotes (') within a message. As a replacement the variable %sq% is available
  which finally gets replaced by a single quote again.
- Since v1.9 the option "pluginmetrics" is no longer available. If you still want to decide if PluginMetrics is allowed
  to send stats you can do that in the "PluginMetrics" folder in your plugin directory.
- If you're using the boss bar you have to regard that messages must not be longer than 64 characters. Otherwise they are automatically
  going to get cut by BarAPI to prevent a Minecraft client crash.
- If an update is available it's recommended to install it. It's guaranteed that it works perfectly fine with every Minecraft version
  (there may be some version dependent functions). Furthermore there are often bug fixes as well as enhancements/new functions included.

If there are still any obscurities, questions ore problems you're always able to contact me on Twitter (@SBPlugin), on BukkitDev (KingDome24)
or via email (contact@simplebroadcast.net) to ask for help. If you find any issues, please report them on GitHub (https://github.com/SimpleBroadcast/SimpleBroadcast/issues).

Hallo lieber SimpleBroadcast User!

Zu Beginn: Danke, dass du mein Plugin verwendest.
In dieser Datei werden einige, wichtige Dinge angesprochen, die man bei der Verwendung dieses Plugins beachten sollte:

- Der Präfix, Suffix sowie alle Nachrichten müssen innerhalb von einzelnen Anführungszeichen stehen ('...'), damit
  die Farbcodes sowie Variablen problemlos funktionieren können.
- Es ist nicht möglich, einzelne Anführungszeichen (') in Nachrichten auszuschreiben. Als Ersatz steht die Variable "%sq%"
  zur Verfügung, die schlussendlich wieder in ein Anführungszeichen umgewandelt wird.
- Seit v1.9 steht die Option "pluginmetrics" nicht mehr direkt zur Verfügung. Wer dennoch darüber bestimmen will, ob PluginMetrics
  Statistiken senden darf, kann dies im globalen "PluginMetrics" - Ordner im Plugin-Verzeichnis tun.
- Sollte die Bossbar verwendet werden, ist zu beachten, dass Nachrichten nicht länger als 64 Zeichen lang sein dürfen, da sie ansonsten
  automatisch von BarAPI gekürzt werden, um einen Absturz des Minecraft-Clients zu verhindern.
- Sollte ein Update vorhanden sein, so ist es sehr empfehlenswert, dieses zu installieren. Es wird garantiert, dass es mit allen Minecraft
  Versionen kompatibel ist (einzelne Funktionen können jedoch versionsspezifisch sein). Zudem bringt es oftmals Fehlerbehebungen sowie
  Verbesserungen bzw. neue Funktionen mit sich.

Sollten weiterhin Unklarheiten, Fragen oder Probleme bestehen, kann man jederzeit auf Twitter (@SBPlugin), auf BukkitDev (KingDome24)
oder via E-Mail (contact@simplebroadcast.net) Kontakt aufnehmen und nach Hilfe fragen. Solltest du einen Fehler entdeckt haben,
so melde diesen bitte auf GitHub (https://github.com/SimpleBroadcast/SimpleBroadcast/issues).
