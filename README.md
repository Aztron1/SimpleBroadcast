SimpleBroadcast
===============

SimpleBroadcast is the easy way for Minecraft servers to broadcast predefined messages with various options.

###Supported server softwares

Software | Status
--- | ---
Bukkit | Supported
Spigot | Supported
Sponge | *Not tested*
Glowstone | Supported *with restrictions*
